# YAG System Diagnostics

Простой Bash-скрипт для сбора информации о системе и генерации отчёта в HTML.  

## Возможности

- система (ядро, аптайм, дата);
- железо (CPU, память, диски, PCI);
- сеть (IP, маршруты, DNS);
- установленные пакеты;
- антивирусы (ClamAV, Kaspersky, Dr.Web, ESET и др.);
- NVIDIA (если есть драйверы);
- службы, процессы, пользователи, группы;
- логи и dmesg.

## Использование

chmod +x yag.sh
./yag.sh
